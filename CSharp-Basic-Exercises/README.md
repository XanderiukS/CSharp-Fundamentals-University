# Fundamentos de Lógica en C# y Windows Forms

Este repositorio contiene una aplicación de escritorio desarrollada en **C#** utilizando **Windows Forms**. Su propósito principal es demostrar la resolución de algoritmos básicos y la aplicación de estructuras de control, ciclos iterativos y condicionales lógicos.

## 🚀 Tecnologías Utilizadas
* C#
* .NET (Windows Forms)
* Visual Studio

## 📋 Ejercicios Resueltos
El proyecto está compuesto por una interfaz gráfica que da solución a los siguientes 6 ejercicios de lógica de programación:

1. **Clasificación de Teléfonos:** Determina si un número es fijo/local, móvil o internacional basándose en la cantidad de dígitos ingresados utilizando `.Length` y condicionales.
2. **Validador de Caracteres:** Identifica si una letra ingresada es una vocal o una consonante mediante una estructura de selección múltiple (`switch`).
3. **Sumatoria de Rango:** Calcula y muestra la suma total de todos los números comprendidos entre el 100 y el 200 haciendo uso del ciclo `while`.
4. **Cuenta Regresiva:** Genera y concatena una secuencia numérica decreciente del 10 al 1.
5. **Buscador de Números Pares:** Encuentra e imprime todos los números pares que existen entre un rango de dos números ingresados por el usuario, utilizando el operador módulo (`%`).
6. **Sistema de Calificaciones:** Evalúa una nota numérica (0-100) y le asigna un nivel de rendimiento textual (Inaceptable, Aceptable, Meritorio) usando operadores lógicos combinados (`&&`).

## 💻 Cómo ejecutar el proyecto
1. Clona este repositorio en tu entorno local.
2. Abre el archivo de la solución (`.sln`) utilizando **Visual Studio**.
3. Presiona el botón de **Iniciar** (o la tecla `F5`) para compilar y ejecutar la interfaz gráfica.
