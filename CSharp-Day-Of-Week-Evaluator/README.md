# Repaso de Funciones y Estructuras Switch en C#

Este repositorio contiene una pequeña aplicación de escritorio en **C#** (Windows Forms) desarrollada como práctica para repasar el uso de funciones con retorno de valores y estructuras de control de flujo.

## 🚀 Tecnologías Utilizadas
* C#
* .NET (Windows Forms)
* Visual Studio

## 📋 Descripción del Proyecto
El programa consiste en una interfaz gráfica sencilla que permite al usuario ingresar un valor numérico (del 1 al 7) y, mediante el uso de un botón, muestra el día de la semana correspondiente. 

El código fuente destaca por la implementación de los siguientes conceptos:
* **Métodos con retorno:** Creación de la función `RetornoDiaSemana` que procesa un número entero y devuelve una cadena de texto (`string`).
* **Estructura Switch:** Uso de casos múltiples (`case`) para evaluar la variable ingresada y asignar el día correcto, incluyendo un caso por defecto (`default`) para manejar números fuera del rango.
* **Conversión de tipos:** Transformación del texto ingresado en la interfaz a un valor entero utilizando `Convert.ToInt32()`.

## 💻 Cómo ejecutar el proyecto
1. Clona este repositorio en tu computadora.
2. Abre la solución (`.sln`) en **Visual Studio**.
3. Presiona el botón de **Iniciar** (o la tecla `F5`) para compilar y probar la aplicación.
